# Social Media Sentiment Analysis Dashboard

## Project Overview
This project performs sentiment analysis on social media data using Python, NLP, Machine Learning, and Power BI.

The system processes social media posts, analyzes sentiments using NLP techniques and AI-based sentiment analysis models, and visualizes insights through interactive Power BI dashboards.

---

# Features

- Data Cleaning & Preprocessing
- NLP Text Processing
- Tokenization
- Stopword Removal
- Stemming
- Lemmatization
- Sentiment Analysis using VADER
- Sentiment Analysis using TextBlob
- Machine Learning using Logistic Regression
- Sentiment Accuracy Evaluation
- Confusion Matrix Visualization
- Interactive Power BI Dashboard
